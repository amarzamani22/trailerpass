Drop your image files into this folder.

Expected filenames (referenced in index.html):

  logo.png           Car Medic logo for the header
                     - Square is best (e.g. 256x256 or 512x512)
                     - PNG with transparent background looks cleanest
                     - You can copy the same logo.png from the Towing Pass folder

Recommended size: 256x256 or larger. PNG with transparent background.

Note: The Trailer Pass shows photos for each vehicle as the driver loads them.
Those photos are uploaded by the driver in real-time and don't need to be
pre-placed here. They live only in the running session.
